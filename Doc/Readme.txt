ขั้นตอนการ Run project
1. ทำการลงโปรแรม python จาก http://www.python.org/download/
2. ลงโปรแกรม django จาก http://www.djangoproject.com/download/.
3. เปิด Terminal กด Alt+Shift 
4. พิมคำสั่งเปิด project เข้าไปใน file ของ project ที่มีหน้า manage.py
5. พิมคำสั่ง python manage.py runserver
6. จากนั้น ทำการใช้งาน project
